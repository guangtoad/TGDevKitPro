## Installation

TG-ObjC is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'TG-ObjC' ,:podspec => 'https://gitee.com/guangtoad/TG-ObjC/raw/master/TG-ObjC.podspec'
```
